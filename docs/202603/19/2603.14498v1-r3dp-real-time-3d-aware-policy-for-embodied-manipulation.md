## 摘要
具身操作需要对物体及其空间关系进行准确的 3D 理解，以规划和执行涉及大量接触的动作。虽然大规模 3D 视觉模型提供了强大的先验知识，但其计算开销会导致实时控制产生不可接受的延迟。我们提出了实时 3D 感知策略（R3DP），在不牺牲实时性能的前提下，将强大的 3D 先验集成到操作策略中。R3DP 的核心创新是异步快慢协作模块，它在不损害实时性能的情况下，将大规模 3D 先验无缝集成到策略中。该系统通过仅在稀疏关键帧上查询预训练的慢速系统（VGGT），同时采用轻量级时间特征预测网络（TFPNet）预测所有中间帧的特征，从而保持实时效率。通过利用历史数据挖掘时间相关性，TFPNet 通过一致的特征估计显著提高了任务成功率。此外，为了实现更有效的多视图融合，我们引入了多视图特征融合器（MVFF），通过显式结合相机内参和外参来聚合跨视图特征。R3DP 为将大模型集成到实时推理系统提供了一种即插即用的解决方案。我们在不同的视觉配置下针对多个基准评估了 R3DP。R3DP 有效地利用了大规模 3D 先验并取得了优异的结果，在平均成功率上分别比单视图和多视图 DP 高出 32.9% 和 51.4%。此外，通过将繁重的 3D 推理与策略执行解耦，与朴素的 DP+VGGT 集成相比，R3DP 的推理时间减少了 44.8%。

## Abstract
Embodied manipulation requires accurate 3D understanding of objects and their spatial relations to plan and execute contact-rich actions. While large-scale 3D vision models provide strong priors, their computational cost incurs prohibitive latency for real-time control. We propose Real-time 3D-aware Policy (R3DP), which integrates powerful 3D priors into manipulation policies without sacrificing real-time performance. A core innovation of R3DP is the asynchronous fast-slow collaboration module, which seamlessly integrates large-scale 3D priors into the policy without compromising real-time performance. The system maintains real-time efficiency by querying the pre-trained slow system (VGGT) only on sparse key frames, while simultaneously employing a lightweight Temporal Feature Prediction Network (TFPNet) to predict features for all intermediate frames. By leveraging historical data to exploit temporal correlations, TFPNet explicitly improves task success rates through consistent feature estimation. Additionally, to enable more effective multi-view fusion, we introduce a Multi-View Feature Fuser (MVFF) that aggregates features across views by explicitly incorporating camera intrinsics and extrinsics. R3DP offers a plug-and-play solution for integrating large models into real-time inference systems. We evaluate R3DP against multiple baselines across different visual configurations. R3DP effectively harnesses large-scale 3D priors to achieve superior results, outperforming single-view and multi-view DP by 32.9% and 51.4% in average success rate, respectively. Furthermore, by decoupling heavy 3D reasoning from policy execution, R3DP achieves a 44.8% reduction in inference time compared to a naive DP+VGGT integration.

*   **研究背景**：
    *   **2D 策略的局限**：传统的基于 2D 图像的模仿学习（如 Diffusion Policy）缺乏显式的几何感知，在处理遮挡、深度判断和复杂空间任务时表现不佳。

*   **研究动机**：开发一种既能利用大规模 3D 基础模型的强空间先验，又能保持实时推理能力的“即插即用”型策略框架。

### 2. 方法论
R3DP 的核心思想是**“解耦 3D 推理与动作执行”**，通过异步架构平衡精度与速度。
*   **异步快慢协作模块 (AFSC)**：
    *   **慢速系统 (Slow System)**：定期（每隔 $\tau$ 帧）调用预训练的 3D 基础模型（如 VGGT），提取高保真但高延迟的 3D 特征。
    *   **快速系统 (Fast System)**：使用轻量级的 **TFPNet**。在非关键帧期间，利用历史特征和当前 2D 图像快速预测当前的 3D 特征。
*   **时间特征预测网络 (TFPNet)**：
    *   采用 DINOv2 作为视觉编码器，结合交替注意力机制（Alternating-Attention）和交叉注意力机制。
    *   通过蒸馏（Distillation）学习 VGGT 的特征分布，确保特征在时间轴上的连续性和一致性。
*   **多视图特征融合器 (MVFF)**：
    *   **显式几何注入**：不同于简单的特征拼接，MVFF 利用相机内参和外参，通过 **PRoPE**（投影位置编码）将多视角特征对齐到统一的空间表示中。
*   **策略头**：将融合后的 3D 感知特征输入到扩散策略（Diffusion Policy）中，生成动作序列。

### 6. 主要结论与发现
*   **性能大幅提升**：R3DP 在仿真环境下的平均成功率达到 69.0%，比单视图 DP 高出 **32.9%**，比多视图 DP 高出 **51.4%**。

*   **几何鲁棒性**：在真机实验中，R3DP 在处理透明物体（玻璃杯）和复杂光照环境时显著优于依赖深度传感器的 DP3，证明了 3D 基础模型先验的优越性。
*   **多视角融合的必要性**：实验显示简单的多视角拼接（DP-multi）效果甚至不如单视角，而显式引入相机几何参数（MVFF）是提升性能的关键。

### 7. 优点
*   **架构创新**：异步快慢协作（AFSC）巧妙解决了“大模型强感知”与“机器人实时性”之间的固有矛盾。
*   **即插即用**：该框架不绑定特定的策略头，可以轻松集成到现有的模仿学习或强化学习算法中。
*   **无需深度传感器**：仅靠 RGB 输入即可获得高质量 3D 感知，降低了硬件成本并提升了对极端材质（透明/反光）的适应性。
*   **时间一致性**：TFPNet 不仅加速了推理，还通过历史信息平滑了特征波动，增强了长程操作的稳定性。

### 8. 不足与局限
*   **预训练依赖**：TFPNet 的效果高度依赖于慢速系统（VGGT）的质量，若基础模型在某些极端场景下失效，预测网络也难以修正。
*   **任务特定预训练**：虽然 TFPNet 是轻量级的，但在新任务或新领域上可能仍需要一定的特征蒸馏预训练时间。
*   **算力门槛**：尽管推理速度快，但训练阶段仍需要较多高性能 GPU（如 A800）支持特征蒸馏。
*   **动态环境挑战**：在物体运动极快的场景下，较大的 $\tau$ 值可能会导致 TFPNet 预测的特征滞后，论文对极端动态场景的讨论较少。
