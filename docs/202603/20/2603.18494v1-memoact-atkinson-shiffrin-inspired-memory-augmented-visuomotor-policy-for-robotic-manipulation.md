## 摘要
记忆增强的机器人策略在处理依赖记忆的任务中至关重要。然而，现有方法通常依赖于简单的观测窗口扩展，难以同时实现精确的任务状态追踪和鲁棒的长时程记忆保留。为了克服这些挑战，受 Atkinson-Shiffrin 记忆模型的启发，我们提出了 MemoAct，这是一种基于分层记忆的策略，利用不同的记忆层级来解决特定的瓶颈。具体而言，无损短期记忆确保了精确的任务状态追踪，而压缩长期记忆则实现了鲁棒的长时程记忆保留。为了丰富评估维度，我们基于 RoboTwin 2.0 构建了 MemoryRTBench，专门用于评估策略在任务状态追踪和长时程记忆保留方面的能力。在模拟和真实场景中进行的大量实验表明，与现有的马尔可夫基准和历史感知策略相比，MemoAct 实现了更优越的性能。项目页面已发布。

## Abstract
Memory-augmented robotic policies are essential in handling memory-dependent tasks. However, existing approaches typically rely on simple observation window extensions, struggling to simultaneously achieve precise task state tracking and robust long-horizon retention. To overcome these challenges, inspired by the Atkinson-Shiffrin memory model, we propose MemoAct, a hierarchical memory-based policy that leverages distinct memory tiers to tackle specific bottlenecks. Specifically, lossless short-term memory ensures precise task state tracking, while compressed long-term memory enables robust long-horizon retention. To enrich the evaluation landscape, we construct MemoryRTBench based on RoboTwin 2.0, specifically tailored to assess policy capabilities in task state tracking and long-horizon retention. Extensive experiments across simulated and real-world scenarios demonstrate that MemoAct achieves superior performance compared to both existing Markovian baselines and history-aware policies. ### 2. 方法论
MemoAct 由三个核心模块组成：
*   **感觉蒸馏模块（Sensory Distillation）**：
    *   使用预训练且冻结的 **DINOv2** 提取图像特征。
    *   通过一个可学习的查询 Token（Query Token）将冗余的视觉特征压缩为高保真的“感觉记忆”。
    *   融合机器人自身的本体感受状态（Proprioception）。
*   **长短期记忆模块（Long Short-Term Memory Module）**：
    *   **短期记忆库（STMB）**：采用无损的固定大小队列（容量 $T_s=6$），确保近期任务流的精确追踪。
    *   **长期记忆库（LTMB）**：存储压缩后的历史信息（容量 $T_k=8$）。
    *   **记忆整合机制**：当 STMB 满时，最旧的 Token 通过因果 Transformer 编码器压缩为一个“摘要 Token”并移入 LTMB；当 LTMB 满时，通过计算相似度合并最接近的相邻 Token 以腾出空间。
    *   **门控融合**：利用交叉注意力机制检索历史，并通过门控网络（Gating Network）自适应融合当前观测与历史背景。
*   **动作解码器**：
    *   采用基于 **扩散模型（Diffusion Policy）** 的解码器，以记忆增强后的 Token 为条件，迭代去噪生成动作轨迹。

### 6. 主要结论与发现
*   **性能领先**：MemoAct 在所有基准测试中均显著优于现有方法。在 MemoryRTBench 上平均成功率达到 96.5%，远超 MVMP (72%) 和 SAMP (58%)。
*   **分工明确**：短期记忆对防止任务步骤跳过或循环（状态追踪）至关重要；长期记忆对找回任务初期的物体状态（长程保留）必不可少。
*   **克服感知别名**：通过层级记忆，机器人能够区分视觉上完全相同的不同任务阶段。

### 7. 优点
*   **理论支撑强**：借鉴成熟的认知心理学模型，解决了机器人领域长期存在的“历史信息如何有效存储”的问题。
*   **架构优雅**：结合了无损短期记忆和压缩长期记忆，在计算效率和信息完整性之间取得了极佳平衡。
*   **高适配性**：作为一种通用的记忆增强插件，可以轻松集成到不同的策略框架（如 DP3）中。
*   **开源贡献**：提供了专门针对记忆能力评估的新基准 MemoryRTBench。

### 8. 不足与局限
*   **视觉精度损失**：为了节省内存，将整张图像压缩为单个 Token，这在需要极高空间精度的微小操作任务（如 Observe and Pick Up）中会导致失败。
*   **窗口限制**：虽然比 FIFO 强，但本质上仍受限于预设的记忆库容量，无法处理无限长度的任务流。
*   **终身学习缺失**：目前的方法主要针对单次任务的记忆，尚未实现将历史经验转化为永久性、可重复利用的知识库。
