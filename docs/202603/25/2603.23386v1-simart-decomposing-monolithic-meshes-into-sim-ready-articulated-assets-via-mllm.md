## 摘要
高质量的关节 3D 资产对于具身智能和物理仿真至关重要，然而 3D 生成目前仍侧重于静态网格，在“仿真就绪”的交互式物体方面仍存在空白。最近的大多数关节物体创建方法依赖于多阶段流水线，这些流水线会在解耦的模块之间累积误差。相比之下，统一的多模态大语言模型（MLLM）提供了一条单阶段路径，用于联合静态资产理解和仿真就绪资产生成。然而，基于密集体素的 3D 标记化会产生冗长的 3D 标记序列和高内存开销，限制了其在复杂关节物体上的可扩展性。为了解决这一问题，我们提出了 SIMART，这是一个统一的 MLLM 框架，可联合执行部件级分解和运动学预测。通过引入稀疏 3D VQ-VAE，SIMART 与密集体素标记相比减少了 70% 的标记数量，从而实现了高保真度的多部件装配。SIMART 在 PartNet-Mobility 和真实场景 AIGC 数据集上达到了最先进的性能，并支持基于物理的机器人仿真。

## Abstract
High-quality articulated 3D assets are indispensable for embodied AI and physical simulation, yet 3D generation still focuses on static meshes, leaving a gap in "sim-ready" interactive objects. Most recent articulated object creation methods rely on multi-stage pipelines that accumulate errors across decoupled modules. Alternatively, unified MLLMs offer a single-stage path to joint static asset understanding and sim-ready asset generation. However dense voxel-based 3D tokenization yields long 3D token sequences and high memory overhead, limiting scalability to complex articulated objects. To address this, we propose SIMART, a unified MLLM framework that jointly performs part-level decomposition and kinematic prediction. By introducing a Sparse 3D VQ-VAE, SIMART reduces token counts by 70% vs. dense voxel tokens, enabling high-fidelity multi-part assemblies. SIMART achieves state-of-the-art performance on PartNet-Mobility and in-the-wild AIGC datasets, and enables physics-based robotic simulation.

*   **核心问题**：如何高效地将静态、单体的 3D 网格（Monolithic Meshes）转化为具备运动学逻辑（如关节、轴向、限制）和物理属性的“仿真就绪”（Sim-ready）关节资产。
*   **研究动机**：
    *   **具身智能需求**：机器人仿真和物理交互需要物体具有可活动的部件，但现有的 3D 生成模型大多只生成静态网格。
    *   **现有方法缺陷**：传统多阶段流水线（分解、推理、组装）容易产生误差累积；而现有的 3D MLLM 方案通常采用密集体素标记化（Voxel Tokenization），导致 Token 序列过长、内存开销巨大，难以处理复杂的关节物体。

### 2. 方法论
*   **核心思想**：提出一个统一的多模态大语言模型（MLLM）框架，通过引入**稀疏 3D VQ-VAE**，在大幅降低计算开销的同时，实现部件分解与运动学参数的同步预测。
*   **关键技术细节**：
    *   **Sparse 3D VQ-VAE**：这是本文的核心创新。它仅对被占据的表面体素进行编码，并引入特殊的“零标记”（Zero Token）来代表空余空间。相比密集体素表示，该方法减少了约 **70%** 的 Token 数量。
    *   **统一 MLLM 骨干**：采用 **Qwen3-VL-8B** 作为核心推理引擎，输入包括视觉图像、稀疏几何 Token 和文本指令，输出结构化的 JSON 数据（包含 URDF 元数据和部件 Token）。
    *   **仿真就绪处理流程**：
        1.  **几何编码**：将 3D 网格离散化为体素并提取稀疏 Token。
        2.  **多模态推理**：MLLM 预测部件分割、关节类型（固定、旋转、平移等）、轴线位置及物理属性（摩擦力、密度等）。
        3.  **后处理分割**：利用基于图的表面分割算法，将预测的体素种子映射回原始高保真网格，确保几何精度。

*   **充分性评价**：实验设计较为充分，特别是引入了 AI 生成的物体作为测试集，证明了模型在处理非标准、非结构化网格时的鲁棒性。对比实验涵盖了当前最前沿的生成式和重建式基准，评价标准客观。

### 6. 主要结论与发现
*   **性能领先**：SIMART 在所有关键指标上均显著优于现有 SOTA 方法，尤其是在处理 AI 生成的复杂物体时，IoU 和几何保真度提升巨大。
*   **效率提升**：稀疏化策略成功解决了 MLLM 处理 3D 数据时的内存溢出（OOM）问题，使得模型能够处理具有更多部件的复杂物体。
*   **下游应用**：生成的资产可以直接导入 NVIDIA Isaac Sim 等仿真器，支持机器人抓取和交互任务，并能应用于 VR/AR 场景。

### 7. 优点
*   **端到端统一性**：将几何理解、部件分割和运动学推理整合在一个模型中，避免了多阶段误差。
*   **高效的稀疏表示**：通过 Sparse VQ-VAE 解决了 3D Token 冗余问题，是 3D MLLM 领域的重要技术改进。
*   **高保真度**：通过将预测结果映射回原始网格，保留了输入物体的精细纹理和几何细节。

### 8. 不足与局限
*   **数据依赖性**：尽管引入了 AI 生成数据，但高质量关节物体的标注数据仍然稀缺，这限制了模型在更广泛、更罕见类别上的泛化能力。
*   **计算成本**：虽然推理效率提升，但微调 8B 规模的 MLLM 仍需要大量的计算资源（32 台 A100）。
*   **复杂运动限制**：目前主要针对常见的机械关节（旋转、平移），对于更复杂的非刚体变形或多级联动关节的处理能力尚待进一步验证。
