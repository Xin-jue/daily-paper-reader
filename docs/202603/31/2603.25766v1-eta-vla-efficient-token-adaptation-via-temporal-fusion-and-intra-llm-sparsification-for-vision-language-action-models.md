## 摘要
将视觉-语言-动作（VLA）模型集成到自动驾驶系统中，为解释复杂场景和执行控制指令提供了一个统一的框架。然而，为了实现准确的时序推理，必须纳入历史多视图帧，这带来了沉重的计算负担，主要源于大语言模型（LLM）中自注意力机制的二次方复杂度。为了缓解这一瓶颈，我们提出了 ETA-VLA，一种用于 VLA 模型的高效 Token 适配框架。ETA-VLA 处理过去 $n$ 帧的多视图图像，并引入了一种新颖的 LLM 内部稀疏聚合器（ILSA）。受人类驾驶员注意力分配的启发，ILSA 在文本查询和时序一致性的引导下，动态地识别并剪枝冗余的视觉 Token。具体而言，我们利用文本引导的评分机制以及保持多样性的稀疏化策略来选择关键 Token 的稀疏子集，从而确保对驾驶场景的全面感知。在 NAVSIM v2 上的广泛实验表明，ETA-VLA 实现了与最先进基准模型相当的驾驶性能，同时将计算 FLOPs 降低了约 32%。值得注意的是，我们的方法剪枝了 85% 的视觉 Token，并将推理 FLOPs 降低了 61%，但在 NAVSIM v2 基准测试中仍保留了 94% 的原始准确率。

## Abstract
The integration of Vision-Language-Action (VLA) models into autonomous driving systems offers a unified framework for interpreting complex scenes and executing control commands. However, the necessity to incorporate historical multi-view frames for accurate temporal reasoning imposes a severe computational burden, primarily driven by the quadratic complexity of self-attention mechanisms in Large Language Models (LLMs). To alleviate this bottleneck, we propose ETA-VLA, an Efficient Token Adaptation framework for VLA models. ETA-VLA processes the past $n$ frames of multi-view images and introduces a novel Intra-LLM Sparse Aggregator (ILSA). Drawing inspiration from human driver attention allocation, ILSA dynamically identifies and prunes redundant visual tokens guided by textual queries and temporal consistency. Specifically, we utilize a text-guided scoring mechanism alongside a diversity-preserving sparsification strategy to select a sparse subset of critical tokens, ensuring comprehensive awareness of the driving scene. Extensive experiments on the NAVSIM v2 demonstrate that ETA-VLA achieves driving performance comparable to state-of-the-art baselines while reducing computational FLOPs by approximately 32\%. Notably, our method prunes 85% of visual tokens and reduces inference FLOPs by 61\%, but still retaining 94% of the original accuracy on the NAVSIM v2 benchmark.

#### 1. 核心问题与研究动机
*   **核心问题**：在自动驾驶领域，视觉-语言-动作（VLA）模型需要处理多视图、多帧的历史图像以实现准确的时序推理。然而，大语言模型（LLM）自注意力机制的计算复杂度随序列长度呈二次方增长（$O(L^2)$），导致“Token 膨胀”问题，使得模型难以在车载嵌入式硬件上实时运行。
*   **研究动机**：受人类驾驶员注意力分配机制的启发，研究者希望在保留关键驾驶信息（如障碍物、交通灯）的同时，动态剪枝冗余的视觉 Token，从而在保证驾驶安全性的前提下大幅提升推理效率。

#### 2. 方法论
ETA-VLA 框架由三个核心组件组成：
*   **时序融合模块 (TFM)**：在进入 LLM 之前，利用可学习的时间嵌入（Time Embedding）和时间加权聚合机制，将过去 $n$ 帧的多视图特征压缩为单一的紧凑特征图，有效缓解了 Token 数量随时间线性增长的问题。
*   **LLM 内部稀疏聚合器 (ILSA)**：这是核心创新点，直接在 LLM 内部层进行细粒度 Token 剪枝：
    *   **动态文本锚点选择**：根据视觉 Token 与指令的相关性，动态筛选出对当前驾驶任务最重要的文本 Token 作为“评分者”。
    *   **无 RoPE 语义评分 (RoPE-free Scoring)**：消除旋转位置编码（RoPE）带来的距离偏差，确保视觉 Token 仅根据其与驾驶指令的语义相关性进行评分。
    *   **多样性保持稀疏化策略**：采用两阶段选择，首先根据语义评分保留 Top-K 关键 Token，随后通过计算特征多样性得分“回收”部分被丢弃但具有空间独特性的 Token（如侧后方盲区信息），模拟人类驾驶员既关注前方重点又兼顾四周的习惯。
*   **动作解码器**：将 LLM 输出的稀疏隐藏状态通过轻量级 Transformer 解码器映射为连续的未来轨迹。

*   **数据集与场景**：使用 **NAVSIM v2** 基准测试，包括 **Navtest**（开环评估）和 **Navhard**（具有扰动的伪闭环评估）。
*   **评估指标**：主要指标为 **EPDMS**（扩展预测驾驶员模型得分），综合了碰撞风险、道路合规性、舒适度等 9 个子项。
*   **对比方法**：对比了 Transfuser、VADv2、HydraMDP++、GTRS、DiffusionDrive 等 SOTA（最先进）自动驾驶模型。

*   **实验规模**：论文进行了主基准测试对比、效率分析实验以及详细的消融实验。
*   **充分性与客观性**：

    *   **消融实验**验证了 TFM、ILSA 评分机制、回收策略的有效性，并对比了 Random、HoloV、SparseVLM 等其他剪枝方案。

#### 6. 主要结论与发现
*   **性能领先**：ETA-VLA 在 Navtest 上达到 85.0 EPDMS，接近人类专家水平（90.3），在 Navhard 上也显著优于之前的 SOTA。
*   **高效剪枝**：在减少 32% FLOPs 的情况下，性能甚至略优于全量 Token 的基线模型，说明 ILSA 起到了过滤噪声的作用。
*   **极限压缩**：即使剪枝 85% 的视觉 Token，推理 FLOPs 降低 61%，模型仍能保留 94% 的原始精度。
*   **层级敏感性**：在 LLM 的中间层（如第 4 层）进行剪枝效果最好，过早或过晚剪枝都会导致关键信息丢失。

#### 7. 优点
*   **技术创新**：提出了“无 RoPE 评分”解决了 LLM 内部视觉 Token 评估的偏差问题，设计非常巧妙。
*   **兼顾安全与效率**：通过“多样性回收”机制防止了因过度剪枝导致的盲区感知缺失，这对于自动驾驶至关重要。
*   **即插即用**：ILSA 模块集成在 LLM 内部，能够利用 LLM 自身的注意力权重进行决策，无需额外的复杂评分网络。

#### 8. 不足与局限
*   **数据集单一**：实验主要集中在 NAVSIM v2，虽然该基准很强，但未在 nuScenes 或 Waymo 等其他主流自动驾驶数据集上验证泛化性。
*   **闭环局限性**：Navhard 虽为伪闭环，但与真实的完全闭环仿真或实车测试仍有差距。
*   **硬件依赖**：虽然降低了 FLOPs，但 LLM 本身的显存占用依然较高，在极低功耗的嵌入式芯片上的部署表现尚待观察。
