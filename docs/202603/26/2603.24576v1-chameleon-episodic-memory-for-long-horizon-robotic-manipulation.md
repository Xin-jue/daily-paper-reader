## 摘要
机器人操作通常需要记忆：遮挡和状态变化可能导致决策时的观测产生感知混淆（perceptually aliased），使得动作选择在观测层面呈现非马尔可夫性，因为相同的观测可能源于不同的交互历史。大多数具身智能体通过语义压缩的轨迹和基于相似性的检索来实现记忆，这会丢弃用于消除歧义的细粒度感知线索，并可能返回感知上相似但与决策无关的情景。受人类情景记忆的启发，我们提出了 Chameleon，它通过写入基于几何的多模态标记（tokens）来保留消除歧义的上下文，并通过可微记忆栈产生目标导向的回忆。我们还引入了 Camo-Dataset，这是一个基于 UR5e 真实机器人的数据集，涵盖了感知混淆下的情景回忆、空间追踪和序列化操作。在各项任务中，Chameleon 在感知易混淆的设置下，相较于强基准模型，一致地提高了决策可靠性和长程控制能力。

## Abstract
Robotic manipulation often requires memory: occlusion and state changes can make decision-time observations perceptually aliased, making action selection non-Markovian at the observation level because the same observation may arise from different interaction histories. Most embodied agents implement memory via semantically compressed traces and similarity-based retrieval, which discards disambiguating fine-grained perceptual cues and can return perceptually similar but decision-irrelevant episodes. Inspired by human episodic memory, we propose Chameleon, which writes geometry-grounded multimodal tokens to preserve disambiguating context and produces goal-directed recall through a differentiable memory stack. We also introduce Camo-Dataset, a real-robot UR5e dataset spanning episodic recall, spatial tracking, and sequential manipulation under perceptual aliasing. Across tasks, Chameleon consistently improves decision reliability and long-horizon control over strong baselines in perceptually confusable settings.

### 1. 核心问题与研究动机
*   **核心问题**：长程机器人操作中的“感知混淆”（Perceptual Aliasing）问题。在复杂任务中，由于遮挡或状态变化，机器人当前的观测可能无法提供足够的信息来决定下一步动作（即非马尔可夫性）。例如，在“三杯球”游戏中，多个杯子外观完全相同，机器人必须依靠历史记忆才能确定球在哪个杯子下。
*   **研究动机**：现有的具身智能体记忆机制通常依赖语义压缩（如文本摘要）或基于相似性的检索（RAG）。这种方式会丢失细粒度的几何线索，且容易检索到视觉相似但决策无关的错误片段。本文旨在模仿人类的“情景记忆”机制，构建一种能够保留细粒度上下文并实现目标导向检索的记忆架构。

### 2. 方法论：Chameleon 架构
Chameleon 采用“感知 $\rightarrow$ 记忆 $\rightarrow$ 策略”的闭环流水线，核心思想是模仿大脑的内嗅皮层（EC）-海马体（HC）-前额叶皮层（PFC）系统。

*   **感知层（Perception）**：
    *   **双流编码**：腹侧流（Ventral）利用预训练的 DINOv2 提取外观特征；背侧流（Dorsal）利用末端执行器（EE）的几何位姿作为锚点，提取空间结构特征。
    *   **几何偏置注意力**：通过对极几何（Epipolar Geometry）约束和 EE 锚点偏置，实现多视角（前视和手眼相机）之间的特征对齐，减少写入记忆时的歧义。
*   **记忆层（Memory）**：
    *   **分层可微栈**：耦合了“情景状态”（慢速、长期存储）和“工作状态”（快速、即时控制）。
    *   **时空锚点**：利用 Mamba 风格的选择性状态空间模型（SSM），将证据写入具有不同半衰期的时空槽位中，实现“模式分离”（Pattern Separation）。
    *   **HoloHead（潜意识想象）**：通过预测未来 2D/3D 路径点来训练记忆状态，强制记忆模块进行“模式补全”（Pattern Completion），确保检索是目标导向的。
*   **策略层（Policy）**：
    *   **条件流匹配（CFM）**：仅基于记忆模块输出的单一决策状态 $h_t$，通过整流流（Rectified Flow）一次性生成未来的末端位姿轨迹，提高长程控制的稳定性。

### 3. 实验设计
*   **数据集与场景**：引入了 **Camo-Dataset**，使用 UR5e 实体机器人收集。包含三个具有挑战性的感知混淆任务：
    1.  **清理指定盘子（情景回忆）**：识别并记住用户放置的特定盘子。
    2.  **三杯球游戏（空间追踪）**：在杯子交换位置后追踪隐藏物体。
    3.  **添加多种调料（顺序操作）**：按特定顺序使用勺子，需记住已完成的步骤。
*   **基准模型（Benchmarks）**：对比了 Diffusion Policy、Flow Matching 和 ACT 等强基线模型。
*   **评估指标**：除了成功率（SR），还提出了决策成功率（DSR，衡量记忆检索准确性）和操作成功率（MSR，衡量动作执行准确性）。

### 4. 资源与算力
*   **硬件**：使用单张 **NVIDIA GeForce RTX 4090 GPU** 进行训练。
*   **训练细节**：每个任务训练 20,000 步，采用 AdamW 优化器，支持 bfloat16 混合精度。
*   **推理延迟**：端到端推理平均延迟为 **82ms**，满足 UR5e 机器人的实时操作需求。

### 5. 实验数量与充分性
*   **实验规模**：每个任务收集 120 个演示视频。在测试阶段，每个类别进行 36 次独立的真实机器人实验。
*   **消融实验**：设计了 5 组消融实验，分别验证了记忆模块、背侧流几何感知、HoloHead 想象目标以及 SSM 结构（对比 Vanilla Mamba）的必要性。
*   **充分性评价**：实验设计涵盖了从感知到决策再到执行的全流程，通过 UMAP 可视化展示了记忆状态的演变，客观地证明了方法在处理非马尔可夫任务时的有效性。

### 6. 主要结论与发现
*   **记忆是关键**：在感知混淆场景下，缺乏有效记忆机制的基线模型（如 Diffusion Policy）表现接近随机猜测，而 Chameleon 实现了极高的决策成功率。
*   **结构化记忆优于简单检索**：基于相似性的记忆库（Memory Bank）在处理视觉相似但逻辑不同的场景时会失效，而 Chameleon 的连续动力学记忆能更好地分离干扰。
*   **几何引导的重要性**：背侧流提供的空间线索显著提升了空间追踪任务（如三杯球）的性能。

### 7. 优点与亮点
*   **生物启发性**：将神经科学中的情景记忆功能（模式分离/补全）成功转化为可计算的工程架构。
*   **解决感知歧义**：针对机器人领域长期存在的非马尔可夫决策难题提供了系统性的解决方案。
*   **细粒度保留**：不同于常见的语义压缩，Chameleon 保留了对操作至关重要的几何和视觉细节。
*   **真实机器人验证**：所有核心结论均在物理实体机器人上得到验证，而非仅停留在仿真环境。

### 8. 不足与局限
*   **泛化性限制**：目前主要在单一机器人平台上验证，跨机器人形态（Cross-embodiment）的迁移能力尚待研究。
*   **缺乏语义先验**：尚未集成大语言模型（LLM）或视觉语言模型（VLM）的高层语义推理能力，在处理开放世界指令时可能受限。
*   **事件分割**：目前依赖预定义的任务阶段信号，未来需要研究如何自动进行事件分割和记忆写入。
