## 摘要
手内物体重定向需要精确的物体位姿估计，以应对复杂的任务动力学。虽然 RGB 感知为位姿跟踪提供了丰富的语义线索，但现有解决方案往往依赖多摄像头设置或高昂的光线追踪成本。我们提出了一种用于单目 RGB 手内重定向的 Sim-to-Real 框架，该框架集成了 3D 高斯泼溅（3DGS）技术以弥合视觉领域的仿真与现实差距。我们的核心见解是在高斯表示空间中执行领域随机化：通过对 3D 高斯应用物理一致的预渲染增强，我们为物体位姿估计生成了逼真的随机化视觉数据。操作策略采用基于课程的强化学习与教师-学生蒸馏技术进行训练，从而实现了复杂行为的高效学习。重要的是，感知和控制模型均可在消费级硬件上独立训练，无需大型计算集群。实验表明，在挑战性的视觉环境下，使用 3DGS 数据训练的位姿估计器优于使用传统渲染数据训练的模型。我们在配备 RGB 摄像头的物理多指灵巧手上验证了该系统，证明了即使在严苛的光照条件下，也能实现对五种不同物体的鲁棒重定向。研究结果表明，高斯泼溅是实现仅依赖 RGB 的灵巧操作的一条切实可行的路径。有关硬件部署视频及更多补充材料，请访问项目网站：https://rffr.leggedrobotics.com/works/viserdex/

## Abstract
In-hand object reorientation requires precise estimation of the object pose to handle complex task dynamics. While RGB sensing offers rich semantic cues for pose tracking, existing solutions rely on multi-camera setups or costly ray tracing. We present a sim-to-real framework for monocular RGB in-hand reorientation that integrates 3D Gaussian Splatting (3DGS) to bridge the visual sim-to-real gap. Our key insight is performing domain randomization in the Gaussian representation space: by applying physically consistent, pre-rendering augmentations to 3D Gaussians, we generate photorealistic, randomized visual data for object pose estimation. The manipulation policy is trained using curriculum-based reinforcement learning with teacher-student distillation, enabling efficient learning of complex behaviors. Importantly, both perception and control models can be trained independently on consumer-grade hardware, eliminating the need for large compute clusters. Experiments show that the pose estimator trained with 3DGS data outperforms those trained using conventional rendering data in challenging visual environments. We validate the system on a physical multi-fingered hand equipped with an RGB camera, demonstrating robust reorientation of five diverse objects even under challenging lighting conditions. Our results highlight Gaussian splatting as a practical path for RGB-only dexterous manipulation. ### ViserDex: 面向鲁棒灵巧手内重定向的视觉 Sim-to-Real 框架总结

#### 1. 核心问题与研究动机
*   **核心问题**：如何在仅使用**单目 RGB 摄像头**的情况下，实现多指灵巧手对复杂几何形状物体的鲁棒手内重定向（In-hand Reorientation）。
*   **研究背景**：
    *   灵巧手操作涉及复杂的动力学和严重的自遮挡，精确的物体位姿估计至关重要。
    *   现有的 Sim-to-Real 方法要么依赖昂贵的多相机阵列或深度/触觉传感器，要么在仿真中使用计算成本极高的光线追踪（Ray Tracing）来弥合视觉差异。
    *   传统的 3D 高斯泼溅（3DGS）虽然渲染快，但通常针对静态场景，且光照与几何耦合，难以直接用于强化学习（RL）中的领域随机化（Domain Randomization）。

#### 2. 方法论
ViserDex 采用模块化设计，将任务分解为感知与控制两个独立训练的阶段：
*   **核心思想**：利用 3DGS 的显式表示特性，在**预渲染阶段（Pre-rasterization）**对高斯属性进行增强，从而生成具有物理一致性且高度多样化的合成数据。
*   **关键技术细节**：
    1.  **3DGS 预渲染增强**：通过对高斯球的球谐函数（SH）系数进行扰动，模拟光照、材质和环境变化。增强策略包括：
        *   **随机噪声**：模拟传感器伪影。
        *   **空间聚类（K-means）**：模拟局部阴影或污渍。
        *   **颜色聚类**：模拟特定材质的光学反应。
        *   **全局偏移**：模拟环境亮度、色温和曝光。
    2.  **控制策略训练（教师-学生架构）**：
        *   **特权教师（Teacher）**：在仿真中利用完整状态信息（速度、接触力等）通过 PPO 算法学习操作技能。引入了基于性能的课程学习（Success-based Curriculum）。
        *   **递归学生（Student）**：通过蒸馏学习，仅利用带有噪声的关节状态和视觉位姿估计。使用递归信念编码器（Belief Encoder）处理部分可观测性。
    3.  **位姿估计器**：基于 ResNet-34 的关键点回归网络，预测物体的 2.5D 坐标，并通过 Rigid Procrustes 算法恢复 6D 位姿。

#### 3. 实验设计
*   **实验场景与对象**：在 16 自由度的 Allegro Hand 上进行验证。测试了 5 种物体：立方体（Cube）、3D 打印玩具（Toy）、橡皮鸭（Duck）、药瓶（Bottle）、地球仪（Globe）。
*   **Benchmark 与对比方法**：
    *   **感知对比**：对比了标准平铺渲染（Standard Tiled）、带领域随机化的平铺渲染（DR Tiled）以及原生 3DGS（Naive GS）。
    *   **光照条件**：分为常规光照（Nominal）和对抗性光照（Adversarial，低光、动态色调切换）。
    *   **控制对比**：与经典的 DeXtreme 框架进行对比。
*   **评估指标**：平均模型点距离（ADD）、位姿精度（<10mm & <10°）、连续成功次数（CS）。

#### 4. 资源与算力
*   **硬件**：使用了单台工作站，配备 Intel i9 CPU 和 **NVIDIA RTX 6000 Ada / RTX 4090 GPU**。
*   **训练时长**：
    *   **教师策略**：简单物体约 26 小时，复杂物体约 90 小时（单/双 GPU）。
    *   **学生蒸馏**：约 16 小时。
    *   **感知模型**：在消费级硬件上独立训练。
*   **效率提升**：相比于之前需要 8 张 A40 GPU 运行 60 小时的方法（如 DeXtreme），ViserDex 在显存效率和训练时间上有显著优化。

#### 5. 实验数量与充分性
*   **实验规模**：
    *   进行了大量的仿真评估（每组 256 个随机环境）。
    *   真实世界部署中，每个物体在两种光照条件下各进行 5 次测试。
    *   针对 3DGS 的四种增强策略进行了详尽的**消融实验（Leave-one-out）**。
*   **客观性与公平性**：实验涵盖了从简单几何体到非凸复杂物体的跨度，并引入了极端的对抗性光照，验证了系统的边界性能。对比实验中保持了网络架构和超参数的一致性。

#### 6. 主要结论与发现
*   **3DGS 的优越性**：基于 3DGS 增强数据训练的位姿估计器在常规和对抗光照下均显著优于传统渲染方法。
*   **感知是瓶颈**：研究发现，真实世界灵巧操作的失败往往源于感知精度下降（尤其是旋转误差），而非控制策略本身。
*   **全局偏移的重要性**：消融实验证明，模拟全局环境变化（Global Shift）是弥合视觉 Sim-to-Real 差距最关键的因素。
*   **鲁棒性**：系统在真实硬件上实现了平均超过 25 次的连续重定向成功。

#### 7. 优点
*   **高效性**：摆脱了对大规模计算集群的依赖，在单张消费级 GPU 上即可完成全流程训练。
*   **视觉鲁棒性**：通过 3D 空间中的预渲染增强，解决了传统 2D 图像增强无法模拟的物理光影一致性问题。
*   **模块化**：感知与控制解耦，便于针对不同物体快速适配。

#### 8. 不足与局限
*   **遮挡限制**：尽管具有一定的鲁棒性，但在极端遮挡（手指完全覆盖物体特征）时，单目视觉仍会失效。
*   **物理建模偏差**：对于表面摩擦力极低或具有特殊材质（如药瓶标签）的物体，仿真与现实之间的动力学差距（Friction Gap）依然存在。
*   **泛化性**：目前的方法仍属于**实例级（Instance-specific）**操作，尚未实现对完全未知类别的通用泛化。
*   **传感器融合缺失**：未集成触觉反馈，无法在视觉完全受阻时通过触觉维持操作。
