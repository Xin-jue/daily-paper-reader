## 摘要
我们提出了 3D 锚定前瞻规划（3D-ALP），这是一种用于机器人操作的系统 2（System 2）推理引擎，它将蒙特卡洛树搜索（MCTS）与作为展开预言机（rollout oracle）的 3D 一致性世界模型相结合。与仅根据当前相机帧评估动作的反应式策略不同，3D-ALP 维护一个持久的相机到世界坐标系（c2w）锚点，该锚点在遮挡情况下依然有效，从而能够对不再直接可见的物体位置进行精确的重新规划。在一项需要空间记忆的 5 步序列到达任务（实验 E3）中，3D-ALP 在需要记忆的步骤上实现了 0.650 ± 0.109 的成功率，而贪婪反应式基准仅为 0.006 ± 0.008（Δ=+0.645）；在第 5 步的成功率上，3D-ALP 达到了 0.822，而贪婪基准为 0.000。消融实验（30 个回合，3 个随机种子）表明，树搜索空间记忆是主要的性能驱动因素（+0.533，贡献了 82% 的增益），更深的前瞻规划也带来了额外收益（+0.111，贡献了 17%）。我们还识别并解决了将 UCT-MCTS（应用于树的上置信界算法）应用于连续机器人操作时的四种结构性失效模式。

## Abstract
We present 3D-Anchored Lookahead Planning (3D-ALP), a System 2 reasoning engine for robotic manipulation that combines Monte Carlo Tree Search (MCTS) with a 3D-consistent world model as the rollout oracle. Unlike reactive policies that evaluate actions from the current camera frame only, 3D-ALP maintains a persistent camera-to-world (c2w) anchor that survives occlusion, enabling accurate replanning to object positions that are no longer directly observable. On a 5-step sequential reach task requiring spatial memory (Experiment E3), 3D-ALP achieves 0.650 0.109 success rate on memory-required steps versus 0.006 0.008 for a greedy reactive baseline (Δ=+0.645), while step 5 success reaches 0.822 against 0.000 for greedy. An ablation study (30 episodes, 3 seeds) isolates tree search spatial memory as the primary driver (+0.533, 82% of gain) with additional benefit from deeper lookahead (+0.111, 17%). We also identify and resolve four structural failure modes in applying UCT-MCTS (Upper Confidence Bounds applied to Trees [10]) to continuous robotic manipulation.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现代机器人系统（如 VLA 模型）大多属于“系统 1”反应式策略，仅根据当前相机帧生成动作。这种机制导致机器人缺乏**空间记忆**，一旦目标物体被遮挡或移出视野，机器人就会“忘记”其位置，无法完成需要多步回溯或长程记忆的任务。
*   **研究背景**：在复杂的序列操作中（例如：访问 A、B、C 后回到 A），如果 A 在第 4 步时已不可见，反应式模型往往会失败。论文旨在引入“系统 2”推理引擎，通过规划和持久化 3D 锚点来赋予机器人持久的场景记忆。

### 2. 论文提出的方法论
3D-ALP 结合了蒙特卡洛树搜索（MCTS）和 3D 一致性世界模型，核心组件包括：
*   **持久化 3D 锚点**：通过机器人正向运动学（FK）实时更新相机到世界（c2w）的坐标变换。即使物体被遮挡，其在 3D 空间中的坐标仍保留在 MCTS 搜索树中，不会随视觉消失而重置。
*   **3D 一致性世界模型（InSpatio-WorldFM）**：作为 MCTS 的展开预言机，根据给定的 c2w 查询渲染预测帧，允许机器人在“想象”的 3D 空间中进行前瞻规划。
*   **混合几何-语义评分器**：结合了视觉语言模型（VLM，如 Florence-2）的语义评分和运动学深度惩罚。这解决了 VLM 存在的“深度盲区”问题（即 2D 投影重合但 3D 空间仍有距离的情况）。
*   **针对连续空间的 MCTS 优化（四项修复）**：
    1.  **F1（零动作陷阱）**：通过 Max-Q 值选择并过滤零增量动作，防止机器人原地打转。
    2.  **F2（树深度衰减）**：在根节点移动后递归重置深度，保持前瞻视界。
    3.  **F3（均值惩罚）**：使用 Max-MCTS 代替均值反向传播，避免优秀路径被差的兄弟分支稀释。
    4.  **F4（UCB1 常数调整）**：将探索常数 $c$ 从 1.414 调低至 0.02，以匹配连续运动学分数的量级。

### 6. 主要结论与发现
*   **记忆缺口巨大**：在需要空间记忆的步骤中，反应式基准几乎完全失效（成功率 0.6%），而 3D-ALP 保持了 **65%** 的成功率。

*   **VLM 局限性**：发现现有的 VLM 在处理合成渲染图时缺乏精确的深度感知，必须依靠几何辅助评分。

### 7. 优点
*   **架构解耦**：3D-ALP 是一个模块化的规划原语，世界模型、评分器和搜索引擎均可独立替换。
*   **解决核心痛点**：显式处理了机器人操作中的物体恒存性和遮挡问题，这是迈向长程复杂任务的关键。
*   **算法鲁棒性**：针对连续控制空间对 MCTS 进行了必要的结构化修正，具有较强的工程参考价值。

### 8. 不足与局限
*   **视觉评分瓶颈**：高度依赖几何预言机或混合评分，纯视觉 VLM 评分在 3D 空间中不够精确。
*   **计算开销**：世界模型的渲染速度较慢，限制了 MCTS 在实时任务中的搜索深度和分支因子。
*   **仿真与现实差距**：目前所有实验均在 MuJoCo 仿真中完成，尚未在真实物理机器人上验证（Sim-to-Real）。
*   **应用限制**：目前主要针对静态场景或物体位置可预测的场景，对于动态变化的复杂环境处理能力尚待验证。
