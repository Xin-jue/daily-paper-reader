## 摘要
用于闭环机器人控制的视觉-语言-动作（VLA）模型通常基于马尔可夫假设，这使得它们在需要历史上下文的任务中容易出错。为了引入记忆，现有的 VLA 模型要么从记忆库中检索（这可能受到干扰项的误导），要么扩展帧窗口（其固定长度仍限制了长期记忆的保持）。在本文中，我们介绍了 ReMem-VLA，这是一种配备了两组可学习查询的循环记忆 VLA 模型：用于在连续帧之间传播信息以支持短期记忆的帧级循环记忆查询，以及用于在时间块之间传递上下文以支持长期记忆的块级循环记忆查询。这些查询通过端到端训练来聚合和维持随时间变化的相关上下文，在不增加额外训练或推理成本的情况下隐式地指导模型的决策。此外，为了增强视觉记忆，我们引入了“过去观测预测”作为辅助训练目标。通过广泛的以记忆为中心的仿真和真实机器人实验，我们证明了 ReMem-VLA 在空间、序列、情节、时间和视觉记忆等多个维度上展现出强大的记忆能力。ReMem-VLA 显著优于无记忆 VLA 基准模型 π0.5 和 OpenVLA-OFT，并在依赖记忆的任务上大幅超越了 MemoryVLA。

## Abstract
Vision-language-action (VLA) models for closed-loop robot control are typically cast under the Markov assumption, making them prone to errors on tasks requiring historical context. To incorporate memory, existing VLAs either retrieve from a memory bank, which can be misled by distractors, or extend the frame window, whose fixed horizon still limits long-term retention. In this paper, we introduce ReMem-VLA, a Recurrent Memory VLA model equipped with two sets of learnable queries: frame-level recurrent memory queries for propagating information across consecutive frames to support short-term memory, and chunk-level recurrent memory queries for carrying context across temporal chunks for long-term memory. These queries are trained end-to-end to aggregate and maintain relevant context over time, implicitly guiding the model's decisions without additional training or inference cost. Furthermore, to enhance visual memory, we introduce Past Observation Prediction as an auxiliary training objective. Through extensive memory-centric simulation and real-world robot experiments, we demonstrate that ReMem-VLA exhibits strong memory capabilities across multiple dimensions, including spatial, sequential, episodic, temporal, and visual memory. ReMem-VLA significantly outperforms memory-free VLA baselines $π$0.5 and OpenVLA-OFT and surpasses MemoryVLA on memory-dependent tasks by a large margin.

### 1. 核心问题与整体含义（研究动机和背景）
现有的视觉-语言-动作（VLA）模型在闭环机器人控制中大多遵循**马尔可夫假设**，即仅依赖当前或极短时间的观测来预测动作。这导致模型在处理需要历史上下文的任务（如“将杯子放回原处”或执行多步长序列任务）时经常失败。
*   **现有方案局限：** 记忆库检索（容易受干扰项误导）、扩展帧窗口（受限于固定长度的上下文窗口）、简单循环神经网络（受限于截断反向传播 TBPTT，难以学习长程依赖）。
*   **研究目标：** 赋予 VLA 模型处理空间、序列、情节、时间和视觉记忆的能力，使其能同时兼顾短期操作精度和长期任务进度追踪。

### 2. 方法论：核心思想与关键技术
ReMem-VLA 提出了一种基于**双层循环查询（Dual-Level Recurrent Queries）**的架构：
*   **双层循环机制：**
    *   **帧级循环查询（Frame-level）：** 每一帧更新一次，通过指数移动平均（EMA）传播信息，负责捕捉短期上下文（如维持姿态、近期物体状态）。
    *   **块级循环查询（Chunk-level）：** 每隔 $k$ 帧（一个动作块）更新一次，更新频率较低，衰减更慢，负责维护长期记忆（如初始配置、整体任务进度）。
*   **无梯度循环路径：** 为了克服 TBPTT 无法传递长程梯度的限制，模型采用固定的 EMA 更新路径和冻结的 VLM 主干。模型只需学习“往记忆里写什么”，而“记忆如何传播”由确定的公式保证。
*   **过去观测预测（POP）：** 引入一个辅助任务头，要求模型根据当前记忆重建过去的 RGB 图像，从而增强视觉细节的保留能力。
*   **槽位流式训练（Slot-based Streaming Training）：** 针对变长序列提出的训练范式。在 B 个槽位中并行运行不同情节，每步采样一帧，在情节结束时重置记忆状态，确保了训练的连续性且无跨情节信息泄露。

### 3. 实验设计
*   **仿真场景：** 使用 **MemoryBench** 基准，包含“放回方块”、“重排方块”、“重新打开抽屉”等任务，并额外设计了一个超过 600 帧的**长时任务（Long Horizon Task）**。
*   **真实世界：** 使用 UR5 机械臂执行四类任务：浇花（时间记忆）、舀米（情节记忆）、按顺序按按钮（序列记忆）、水果归位（视觉/空间记忆）。
*   **对比基准（Baselines）：** 
    *   **$\pi$0.5** 和 **OpenVLA-OFT**（无记忆的强力基准）。
    *   **MemoryVLA**（基于检索的记忆模型）。

### 4. 资源与算力
*   **硬件：** 训练使用了 **8 张 NVIDIA A100 GPU**；推理部署在单张 **RTX 4090 GPU** 上。
*   **模型规模：** 采用 Qwen3-VL-2B 作为冻结的视觉语言模型主干。
*   **性能：** 推理频率约为 10Hz，通过复用动作块可提升至近 20Hz。
*   **训练细节：** 总计训练 150k 步，Batch Size 为 64。

### 5. 实验数量与充分性
*   **实验规模：** 仿真中每个任务进行 100 次测试，真实世界每个任务进行 50 次测试。
*   **消融实验：** 针对双层查询的必要性、POP 辅助任务的作用、循环路径是否可学习、EMA 超参数、查询向量数量等进行了详尽的对比。
*   **充分性评价：** 实验覆盖了记忆的五个关键维度（空间、时间、序列、情节、视觉），对比了当前最先进的 VLA 模型，实验设计客观且具有挑战性。

### 6. 主要结论与发现
*   **记忆能力显著提升：** ReMem-VLA 在 MemoryBench 上的平均成功率达到 94.5%，远超 $\pi$0.5（8.25%）和 MemoryVLA（1.5%）。
*   **双层机制的互补性：** 帧级查询对短期操作至关重要，而块级查询是解决长时任务的关键。
*   **无梯度路径的必要性：** 实验证明，在 TBPTT 框架下，让循环路径可学习反而会破坏长时记忆的形成。
*   **视觉增强：** POP 任务显著提升了模型在需要精确视觉回溯任务中的表现。

### 7. 优点（亮点）
*   **架构优雅：** 通过简单的 EMA 和双层查询实现了强大的记忆功能，不增加推理延迟。
*   **训练创新：** 槽位流式训练解决了循环模型在变长机器人数据上的批处理难题。
*   **多维度验证：** 不仅关注空间记忆，还深入探讨了时间长度感知和复杂序列逻辑。

### 8. 不足与局限
*   **泛化性限制：** 该模型尚未在 Open-X-Embodiment 等大规模机器人数据集上进行预训练，其在未见任务上的通用泛化能力有待验证。
*   **主干冻结：** 依赖于冻结的 VLM，可能限制了模型针对特定机器人感知任务进行深度微调的潜力。
*   **任务复杂度：** 虽然解决了记忆问题，但对于极高动态性或需要复杂物理交互理解的任务，仍受限于底层 VLM 的表征能力。
