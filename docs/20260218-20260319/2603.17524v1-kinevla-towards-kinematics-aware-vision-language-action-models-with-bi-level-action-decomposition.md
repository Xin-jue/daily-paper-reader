## 摘要
在本文中，我们引入了一项全新的富含运动学信息的视觉-语言-动作（VLA）任务。在该任务中，语言指令在关键时刻从开始到结束密集地编码了多种运动学属性（如方向、轨迹、朝向和相对位移），这与现有的仅粗略或部分捕捉运动学的动作指令不同，从而支持细粒度和个性化的操作。在这种设定下，任务目标保持不变，而执行轨迹必须适应指令级的运动学规范。为了应对这一挑战，我们提出了 KineVLA，这是一个视觉-语言-动作框架，它通过双层动作表示和双层推理标记（reasoning tokens）显式地将目标层级的不变性与运动学层级的可变性解耦，这些标记作为显式的、受监督的中间变量来对齐语言和动作。为了支持这一任务，我们构建了涵盖仿真和真实世界机器人平台的运动学感知 VLA 数据集，其特点是具有指令级的运动学变化和双层标注。在 LIBERO 和 Realman-75 机器人上进行的大量实验表明，KineVLA 在运动学敏感的基准测试中始终优于强大的 VLA 基线模型，实现了更精确、可控且泛化能力更强的操作行为。

## Abstract
In this paper, we introduce a novel kinematics-rich vision-language-action (VLA) task, in which language commands densely encode diverse kinematic attributes (such as direction, trajectory, orientation, and relative displacement) from initiation through completion, at key moments, unlike existing action instructions that capture kinematics only coarsely or partially, thereby supporting fine-grained and personalized manipulation. In this setting, where task goals remain invariant while execution trajectories must adapt to instruction-level kinematic specifications. To address this challenge, we propose KineVLA, a vision-language-action framework that explicitly decouples goal-level invariance from kinematics-level variability through a bi-level action representation and bi-level reasoning tokens to serve as explicit, supervised intermediate variables that align language and action. To support this task, we construct the kinematics-aware VLA datasets spanning both simulation and real-world robotic platforms, featuring instruction-level kinematic variations and bi-level annotations. Extensive experiments on LIBERO and a Realman-75 robot demonstrate that KineVLA consistently outperforms strong VLA baselines on kinematics-sensitive benchmarks, achieving more precise, controllable, and generalizable manipulation behaviors.

#### 1. 论文的核心问题与整体含义
现有的视觉-语言-动作（VLA）模型（如 OpenVLA, RT-2）主要关注“完成什么任务”（目标导向），而忽略了“如何完成任务”（过程导向）。在现实交互中，同一个目标（如“放好瓶子”）可能对应完全不同的运动学要求（如“标签向左”或“标签向右”）。
**核心问题：** 现有模型对细粒度运动学指令（方向、轨迹、姿态、相对位移）不敏感，无法处理“目标不变但运动轨迹变化”的复杂操控任务。论文旨在通过显式解耦目标与运动学特征，提升机器人操作的精确性与可定制性。

#### 2. 论文提出的方法论
KineVLA 框架的核心思想是**双层解耦（Bi-Level Decomposition）**，具体包含以下关键技术：
*   **双层动作表示（Bi-Level Action Representation）：** 采用残差向量量化变分自编码器（RVQ-VAE）。
    *   **第一层（目标层）：** 捕捉低频、全局的任务结构和语义目标。
    *   **第二层（运动学层）：** 捕捉高频、精细的运动修正和接触细节。
    *   **训练策略：** 两阶段训练。先在多任务数据上预训练目标层码本，再在运动学数据集上冻结目标层并训练运动学层码本。
*   **双层推理标记（Bi-Level Reasoning Tokens）：** 引入类似思维链（CoT）的机制。模型在生成动作前，先生成“粗粒度目标推理”和“细粒度运动学推理”文本，作为对齐语言与动作的中间变量。
*   **互信息正则化（Mutual Information Regularization）：** 使用 InfoNCE 损失函数最大化推理文本与动作标记之间的条件互信息 $I(\text{Reasoning}; \text{Action} | C)$，确保生成的轨迹在语义上与推理逻辑保持一致，减少执行漂移。

#### 6. 论文的主要结论与发现
*   **目标与运动学的解耦是必要的：** 传统 VLA 在目标成功率上表现尚可，但在运动学成功率上存在显著短板；KineVLA 在保持目标成功率的同时，大幅提升了运动学精度。
*   **推理 Token 的引导作用：** 显式的文本推理能显著增强模型对复杂指令的理解，使动作生成更具解释性。
*   **效率优势：** 相比于计算昂贵的扩散模型，基于双层 VQ 的架构在保证精度的同时，维持了较高的实时推理频率。

*   **任务定义新颖：** 首次系统性地提出了“富含运动学信息”的 VLA 任务，贴合个性化机器人服务的实际需求。
*   **架构设计优雅：** 通过双层 RVQ-VAE 巧妙地解决了语义目标与物理细节的冲突。
*   **数据集贡献：** 提供了高质量的、带有双层 CoT 标注的运动学感知数据集，填补了领域空白。

#### 8. 不足与局限
*   **场景局限性：** 目前实验主要集中在桌面操作（Tabletop Manipulation），尚未扩展到全身移动操控或更复杂的动态环境。
*   **坐标系依赖：** 运动学指令依赖于第三人称相机坐标系（左/右/前/后），如果相机视角发生大幅度变化，模型的泛化能力可能受限。
*   **推理长度：** 引入双层推理 Token 会增加输出序列长度，虽然目前延迟可控，但在极高性能要求的实时场景下仍可能存在瓶颈。
